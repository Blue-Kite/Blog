4. 댓글: Giscus

## 배포

배포할 때 항상 Vercel을 사용했다. 사실 Vercel은 알아서 다 해줘서 개발자가 딱히 해야할 게 없었다. 기껏해야 환경변수 추가하기 정도?  
그래서 이번에는 AWS를 사용해보고자 했다. API 연동이 없어서 정적 배포로도 충분할 거 같았다. S3 + CloudFront + Route53 조합을 이용했으며 가비아에서 도메인을 구매했다.
